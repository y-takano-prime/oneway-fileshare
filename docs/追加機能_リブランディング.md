# 実装指示書：リブランディング（AXON / ホワイト×エレクトリックブルー）

## 概要

システム名を `oneway-fileshare` から `AXON` に変更し、UI全体をホワイト×エレクトリックブルー（#0066FF）の未来的デザインに刷新する。

---

## 注意事項

- `CLAUDE.md` の禁止ファイルは一切触れないこと
- `php artisan migrate:fresh` は実行しないこと
- 新規ファイル作成・削除は1件ずつ確認を取ること
- 同じファイルを3回以上修正したら作業を止めて報告すること
- 各項目完了ごとに「項目X完了」と報告してから次へ進むこと

### DL状況サマリー（ダッシュボード上部カード群に追加）

コントローラ側（`DashboardController@index` または `HomeController@index`）で以下を渡す：

```php
$urls = DownloadUrl::query()
    ->when(Auth::user()->role !== 'admin', fn($q) => $q->where('user_id', Auth::id()))
    ->with('sharedFile')
    ->latest()
    ->get();

$totalUrls  = $urls->count();
$doneCount  = $urls->where('download_count', '>', 0)->count();
$waitCount  = $urls->where('download_count', 0)->filter(fn($u) => !$u->expires_at->isPast())->count();
$expCount   = $urls->filter(fn($u) => $u->expires_at->isPast())->count();

$totalSize  = SharedFile::when(Auth::user()->role !== 'admin', fn($q) => $q->where('user_id', Auth::id()))
    ->sum('file_size');
$storageUsedMb = round($totalSize / 1024 / 1024, 1);
$storageCapMb  = 1024; // 1GB上限（必要に応じて変更）
$storagePercent = min(100, round($storageUsedMb / $storageCapMb * 100));
```

Viewに渡す変数：`$totalUrls`, `$doneCount`, `$waitCount`, `$expCount`, `$storageUsedMb`, `$storageCapMb`, `$storagePercent`
