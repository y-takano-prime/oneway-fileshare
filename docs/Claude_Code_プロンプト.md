# Claude Code 一括実装プロンプト

以下をそのまま Claude Code のチャット欄に貼り付けて実行する。

---

## 貼り付けるプロンプト

```
以下のドキュメントをすべて読んでから作業を開始してください。

1. CLAUDE.md（プロジェクト概要・AI向け制約）
2. docs/実装指示書.md（実装手順・禁止事項）
3. docs/設計仕様書.md（詳細設計）
4. docs/要件定義書.md（機能要件）

読み終えたら、以下のルールを必ず守って実装してください。

【必須ルール】
- 作業開始前に git status で現在の状態を確認すること
- CLAUDE.md の「絶対に編集・削除してはいけないファイル」は一切触れないこと
- 「禁止コマンド」は実行しないこと
- 同じファイルを3回以上修正しようとした場合は作業を止めて報告すること
- ファイルの新規作成・削除は1件ずつ確認を取ること
- PHP 7.4 非対応の構文（named arguments・union typesなど）は使用しないこと

【実装手順】
docs/実装指示書.md の Step 1 から Step 9 を順番に実装してください。
各 Step が完了したら「Step X 完了」と報告してから次へ進んでください。
エラーが発生した場合は、3回試みても解決しない場合は作業を止めて報告してください。
```

---

## 注意事項

- プロンプトを貼り付ける前に、必ず git commit で現在の状態をバックアップしておくこと
- Claude Code が作業を開始したら、完了報告が来るまで別の指示を与えないこと
- Step ごとに報告が来たらブラウザで動作確認してから次の Step を許可すると安全
