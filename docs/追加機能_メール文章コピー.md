# 追加機能指示書：送付メール文章コピー機能

## 概要

URL詳細画面（`urls/show.blade.php`）に、相手先へ送付するメール文章を自動生成し、クリップボードにコピーできるセクションを追加する。

担当者はコピーしたテキストをNicollabo等のメールシステムに貼り付けて送信する運用を想定。

---

## 修正対象ファイル

**1ファイルのみ修正：**
`resources/views/urls/show.blade.php`

**新規ファイル作成不要。**

---

## 実装内容

### 追加場所

既存の「URL詳細カード」と「アクセスログカード」の間に、以下のカードを追加する。

```
[URL詳細カード]          ← 既存（変更しない）
[送付メール文章カード]   ← ここに追加
[アクセスログカード]     ← 既存（変更しない）
```

---

### Blade変数 `$mailText` の内容

コントローラーで生成して渡す（下記参照）。

テキストの形式（改行・スペース含めて正確に）：

```
件名：ファイルのご案内

{recipient_email} 様

お世話になっております。
以下のURLよりファイルをダウンロードいただけます。

■ファイル名
{original_name}

■ダウンロードURL
{download_url}

■パスコード
{passcode_hint}

■有効期限
{expires_at}

ダウンロード後は手順に従い、パスコードおよびメール認証を完了してください。

よろしくお願いいたします。
```

変数の対応：
| プレースホルダー | 値 |
|---|---|
| `{recipient_email}` | `$url->recipient_email` |
| `{original_name}` | `$url->sharedFile->original_name` |
| `{download_url}` | `route('download.passcode', $url->token)` |
| `{passcode_hint}` | パスコードが設定されている場合は「別途ご連絡済みのパスコードをご入力ください」、設定なしの場合は「（パスコードなし）」 |
| `{expires_at}` | `$url->expires_at->format('Y年m月d日 H:i')` |

### コントローラーの修正

**ファイル:** `app/Http/Controllers/DownloadUrlController.php`

`show()` メソッドを以下のように修正する：

```php
public function show(DownloadUrl $url)
{
    $url->load(['sharedFile', 'accessLogs' => function ($query) {
        $query->latest();
    }]);

    $passcodeHint = $url->passcode
        ? '別途ご連絡済みのパスコードをご入力ください'
        : '（パスコードなし）';

    $mailText = implode("\n", [
        '件名：ファイルのご案内',
        '',
        $url->recipient_email . ' 様',
        '',
        'お世話になっております。',
        '以下のURLよりファイルをダウンロードいただけます。',
        '',
        '■ファイル名',
        $url->sharedFile->original_name,
        '',
        '■ダウンロードURL',
        route('download.passcode', $url->token),
        '',
        '■パスコード',
        $passcodeHint,
        '',
        '■有効期限',
        $url->expires_at->format('Y年m月d日 H:i'),
        '',
        'ダウンロード後は手順に従い、パスコードおよびメール認証を完了してください。',
        '',
        'よろしくお願いいたします。',
    ]);

    return view('urls.show', [
        'url' => $url,
        'mailText' => $mailText,
    ]);
}
```

### JavaScriptコピー処理

既存の `@section('scripts')` 内に `copyMailText()` 関数を追加する（既存の `copyDownloadUrl()` は残す）：

```javascript
function copyMailText() {
    const textarea = document.getElementById('mail-text');
    textarea.select();
    textarea.setSelectionRange(0, 99999);

    const feedback = document.getElementById('copy-feedback');

    if (navigator.clipboard && window.isSecureContext) {
        navigator.clipboard.writeText(textarea.value).then(() => {
            feedback.classList.remove('d-none');
            setTimeout(() => feedback.classList.add('d-none'), 2000);
        });
    } else {
        document.execCommand('copy');
        feedback.classList.remove('d-none');
        setTimeout(() => feedback.classList.add('d-none'), 2000);
    }
}
```

## 注意事項

- 既存の `copyDownloadUrl()` 関数・URLコピーボタンは**削除しないこと**
- 既存のアクセスログカードは**変更しないこと**
- パスコードの実際の値はセキュリティ上メール本文に含めない（「別途ご連絡済み」の表記にする）
- `CLAUDE.md` の禁止ファイルは一切触れないこと
