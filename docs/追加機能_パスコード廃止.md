# 追加機能指示書：パスコード廃止

## 概要

セキュリティはメールアドレス確認＋OTP（ワンタイムコード）の2段階で担保し、
パスコード入力ステップを廃止する。

ダウンロードフロー変更：
- 変更前：URLアクセス → パスコード入力 → メールアドレス入力 → OTP入力 → ダウンロード
- 変更後：URLアクセス → メールアドレス入力 → OTP入力 → ダウンロード

---

## 変更対象ファイル

### 1. `app/Http/Controllers/DownloadController.php`

- `showPasscode()` メソッド：パスコード入力をスキップし、直接メールアドレス入力画面を表示する
- `verifyPasscode()` メソッド：不要になるが、ルートが残っている間はリダイレクトのみ返すよう修正

`showPasscode()` を以下のように変更する：

```php
public function showPasscode($token)
{
    $url = $this->findUrl($token);

    if ($error = $this->checkExpiry($url)) {
        return $error;
    }

    $this->log($url, 'access');
    $this->markVerified($token, 'passcode_verified'); // スキップ扱いで自動付与

    return view('download.passcode', ['token' => $token, 'step' => 'email']);
}
```

`verifyPasscode()` を以下のように変更する：

```php
public function verifyPasscode($token, Request $request)
{
    return redirect()->route('download.passcode', ['token' => $token]);
}
```

### 2. `resources/views/download/passcode.blade.php`

- `step === 'passcode'` のブロックを削除する
- `step === 'email'` のブロックのみ残す（メールアドレス入力画面）

### 3. `app/Http/Controllers/DownloadUrlController.php`

- `store()` メソッドのパスコードバリデーションを削除する
- `create()` に渡す `$mailText` のパスコード行を削除する

`store()` のバリデーションから以下を削除：

```php
// 削除する
'passcode' => [
    'required',
    'string',
    'min:8',
    'max:32',
    'regex:/^(?=.*[A-Za-z])(?=.*\d).+$/',
],
```

`DownloadUrl::create()` から以下を削除：

```php
// 削除する
'passcode' => Hash::make($validated['passcode']),
```

### 4. `resources/views/urls/create.blade.php`

パスコード入力フィールドを削除する。

### 5. `resources/views/urls/show.blade.php`

### 6. `app/Http/Controllers/DownloadUrlController.php` の `show()` メソッド

## 注意事項

- `database/migrations/` は変更しないこと（`download_urls.passcode` カラムはDBに残したままでよい）
- `CLAUDE.md` の禁止ファイルは一切触れないこと
- `php artisan migrate:fresh` は実行しないこと
- 同じファイルを3回以上修正したら作業を止めて報告すること
- 各項目完了ごとに「項目X完了」と報告してから次へ進むこと
